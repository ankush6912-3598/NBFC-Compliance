# NBFC Compliance AI Model

## Project Overview

The NBFC Compliance AI Model is designed to help Non-Banking Financial Companies (NBFCs) ensure compliance with regulatory norms and guidelines. This project leverages artificial intelligence to automate the processing of compliance-related tasks, thereby reducing human error and increasing efficiency.

## Features
- **Automated Compliance Checking**: Automatically checks documents and reports against regulatory requirements.
- **Real-time Monitoring**: Provides real-time alerts for compliance deadlines and changes in regulations.
- **Data Analytics**: Analyzes compliance data to provide insights and reporting features.
- **User-Friendly Interface**: Intuitive dashboard for easy navigation and management of compliance tasks.

## Setup Instructions
1. **Clone the Repository**:  
   Run the following command to clone the repository:
   ```bash
   git clone https://github.com/ankush6912-3598/NBFC-Compliance.git
   ```
2. **Install Dependencies**:  
   Navigate to the project directory and install the required dependencies:
   ```bash
   cd NBFC-Compliance
   npm install
   ```
3. **Configuration**:  
   Update the configuration files with your environment settings, particularly database and API keys.
4. **Run the Application**:  
   Start the server with the following command:
   ```bash
   npm start
   ```
5. **Access the Application**:  
   Open your web browser and go to `http://localhost:3000` to access the application.
